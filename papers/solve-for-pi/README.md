# Solving for π: Recovering Geometry from Physics

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.19014277.svg)](https://doi.org/10.5281/zenodo.19014277)

**Author:** Jay Andrew Carpenter  
**Published:** [Zenodo](https://doi.org/10.5281/zenodo.19014277) (DOI: 10.5281/zenodo.19014277)

## Summary

The self-consistency equation is reversed so that given only a measured value of α (without π) and the factorial series S, one solves a cubic equation to recover π. Using the Fan et al. (2023) measurement recovers 11 digits of π exactly.

**Keywords:** pi, fine structure constant, cubic equation, geometry from physics, self-consistency
