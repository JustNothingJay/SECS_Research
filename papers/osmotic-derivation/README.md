# Osmotic Selectivity as a Deterministic Function of the Fine Structure Constant

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.19000474.svg)](https://doi.org/10.5281/zenodo.19000474)

**Author:** Jay Andrew Carpenter  
**Published:** [Zenodo](https://doi.org/10.5281/zenodo.19000474) (DOI: 10.5281/zenodo.19000474)

## Summary

A deterministic chain from the fine structure constant α to the osmotic selectivity of aquaporin water channels, through the Bohr radius, O–H bond length, water molecule diameter, and pore width. The fixed point is determined by α.

**Keywords:** osmotic selectivity, aquaporin, fine structure constant, Bohr radius, water channel
