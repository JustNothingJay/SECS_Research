# An Algebraic Value of α⁻¹ Indistinguishable from the Most Precise Measurement in Physics

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.19042747.svg)](https://doi.org/10.5281/zenodo.19042747)

**Author:** Jay Andrew Carpenter  
**Published:** [Zenodo](https://doi.org/10.5281/zenodo.19042747) (DOI: 10.5281/zenodo.19042747)

## Summary

A single algebraic equation containing no empirical inputs produces α⁻¹ = 137.035999177, which falls 11 parts per billion (0.3σ) from the Harvard electron g-2 extraction. At current experimental precision, the two values are indistinguishable.

**Keywords:** fine structure constant, algebraic derivation, electron g-2, precision measurement, self-consistency
