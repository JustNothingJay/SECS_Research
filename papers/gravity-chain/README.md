# Precision Dominoes: From an Algebraic Fine Structure Constant to the Gravitational Constant

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.19047229.svg)](https://doi.org/10.5281/zenodo.19047229)

**Author:** Jay Andrew Carpenter  
**Published:** [Zenodo](https://doi.org/10.5281/zenodo.19047229) (DOI: 10.5281/zenodo.19047229)

## Summary

An algebraic equation with zero empirical inputs produces α⁻¹ = 137.035999177. This paper traces a chain of five algebraic links from that value to the gravitational constant G, the worst-known constant in physics.

**Keywords:** gravitational constant, fine structure constant, precision chain, algebraic derivation, fundamental constants
