# The Edgeless Spreadsheet: Toroidal Self-Reference and the Four-Fold Collapse

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.19029533.svg)](https://doi.org/10.5281/zenodo.19029533)

**Author:** Jay Andrew Carpenter  
**Published:** [Zenodo](https://doi.org/10.5281/zenodo.19029533) (DOI: 10.5281/zenodo.19029533)

## Summary

A spreadsheet with self-referencing cells and no circular references has a toroidal topology rather than flat boundaries. The contraction path passes through exactly four layers at every radial cross-section, each corresponding to one fundamental force.

**Keywords:** toroidal topology, self-reference, four forces, collapse algebra, fixed point
