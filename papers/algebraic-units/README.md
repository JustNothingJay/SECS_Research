# One Measurement Derives Them All

**Algebraic Dimensional Structure from the Eigenvalue Tower**

Jay Carpenter, March 2026

## Summary

Three algebraic equations (for alpha, the proton-electron mass ratio, and the Planck hierarchy) plus five SI definitions and one measured constant (the Rydberg constant) reproduce the entire CODATA table of fundamental constants. Existing unit libraries carry ~20 empirical conversion factors. This framework needs one.
