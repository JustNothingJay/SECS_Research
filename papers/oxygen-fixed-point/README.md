# Oxygen as Fixed Point: Wind, Blood, and the Constraint Surface of Existence

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.19020287.svg)](https://doi.org/10.5281/zenodo.19020287)

**Author:** Jay Andrew Carpenter  
**Published:** [Zenodo](https://doi.org/10.5281/zenodo.19020287) (DOI: 10.5281/zenodo.19020287)

## Summary

Molecular oxygen is maintained at a fixed-point concentration at every scale where chemistry operates — 98% in blood, 20.946% in atmosphere, ~1% of baryonic mass. The same gradient-driven transport mechanism operates at each scale.

**Keywords:** oxygen, fixed point, haemoglobin, atmosphere, constraint surface, scale invariance
