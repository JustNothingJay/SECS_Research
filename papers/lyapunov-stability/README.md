# Vulnerability Windows as Lyapunov Stability Margins: A Dynamical Systems Model of Gestational Oxygen Sensitivity

**Author:** Jay Andrew Carpenter
**Date:** 2026-03-12

## Summary

Each fetal organ system passes through a developmental period during which it is maximally sensitive to oxygen disruption. The General Theory of Latent Gestational Injury identifies these vulnerability windows and maps them to clinical outcomes. This paper provides the dynamical systems formalisation of that mapping.

Lyapunov stability margins quantify the distance from the current developmental trajectory to the nearest attractor boundary, giving a precise measure of how much perturbation each organ system can absorb at each gestational stage before being displaced to a different developmental outcome.

**Keywords:** Lyapunov stability, dynamical systems, vulnerability windows, dissipative structures, gestational programming, far-from-equilibrium, Lotka-Volterra, HIF-1α, preeclampsia, developmental attractor
