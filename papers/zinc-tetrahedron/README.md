# The Shape of Existence: Tetrahedral Zinc and the Gap That Names Its Own Element

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.19032617.svg)](https://doi.org/10.5281/zenodo.19032617)

**Author:** Jay Andrew Carpenter  
**Published:** [Zenodo](https://doi.org/10.5281/zenodo.19032617) (DOI: 10.5281/zenodo.19032617)

## Summary

Biological zinc adopts tetrahedral coordination with four ligands. The tetrahedron has three structural properties connecting it to the eigenvalue tower: integer ratio R/r = 3, a gap filling ~30% of volume, and a 4-cell embryonic stage arranging tetrahedrally as the first enclosed interior.

**Keywords:** zinc, tetrahedron, eigenvalue tower, biological coordination, embryonic geometry
